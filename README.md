# Armed_Forces_Management_System
Implemeting Database Management App to be able to quickly access and modify the data of the armed forces


## Features
* desktop App
* Simple UI interface
* Report Generation

## Technologies
* Tkinter
* ttkbootstrap
* Numpy,Pandas
* Oracle DataBase
